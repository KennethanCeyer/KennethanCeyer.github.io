BJA

(단체 + 모임 + 활동) = 어썸 = 흑마법 = 약팔이

BJA SECURITY

BJA는 보안을 연구하는 소규모 단체입니다.
보안과 관련된 CTF, 해커톤에 참가하여 문제를 풀고 이를 바탕으로 나온 자료를 발표하거나 각종 알려진 취약점 그리고 알려지지 않은 취약점들을 발견하여 리포팅하는 활동을 주로 합니다. 관련된 보안 분야는 리버싱 / 웹보안 / 네트워크보안 / 시스템보안 / 메타스플로잇 / 포렌식 / 암호보안 등 다양합니다.

현재는 단체 활동이 한시적으로 중지된 상태입니다.
앞으로 다시금 선보일 BJA 활동을 기대해주세요!

요약

  1. 해커톤 / CTF 풀이
  2. 취약점 보고 / 기여
  3. 보안과 관련된 전분야 연구

DevTeam KAK

개발의 구조의 다방면의 각(角)을 바라보는 의도로 만들어진 단체입니다.
주로 실력있는 젊은 개발자들을 모아 연구와 기여가 접목된 형태의 프로젝트나 난이도가 요구되는 전문 기술이 필요하는 등의 상황이 발생할 때 합쳐져서 목적을 달성하고 해산하는 팀입니다. DevStudent팀이 학습만을 하는 교육단체라면, KAK은 목적을 이루기위해 잠깐 모였다가 해산하는 단체입니다.

지금은 활동이 중단된 상태지만 NLP, DeepLeaning, DataFlow등의 데이터 연구분야를 위해 조만간 다시 모일 예정입니다.

요약

  1. 비정기적인 모임과 세미나
  2. 특정 카테고리에 대해서 연구

DevStudent

데브 스튜던트는 오로지 학습만을 위해 만들어진 단순한 팀의 형태를 가지고 있습니다.
데브 스튜던트에 함께하기 위해서는 많은 기술적인 배경지식이 요구되지 않습니다. 확실한 자기동기 그리고 꾸준한 커뮤니티 참여 등을 통해 각각의 참여자들이 다양한 기술 키워드들을 살펴보고 알아가며 최종적으로는 어떤 하나의 분야를 완전히 습득하는 것을 목표로합니다.

데브 스튜던트는 기수 형태로 운영되고 각각의 기수마다 서로다른 분야를 학습하고 발표하게 됩니다.


요약

  1. 초보자도 참가가능
  2. 정기적인 모임과 발표
  3. 특정 한 분야에 대해서 학습
